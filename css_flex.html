Flex_레이아웃 속성

flex 레이아웃은 부모자에 flex를 선언하고 자식요소를 유연하게 배치하는 속성(부모자에게 명령어를 걸어줌)

ul > li의 경우 ul에게 flex를 선언하면 li가 가로정렬 된다.
이때 li는 item의 속성을 사용한다.

flex - container 속성
속성명 속성값과 의미
display flex를 선언하면 '자손자를 flex로 정렬하겠다' 라는 선언문을 만든다.
justify-content 수평정렬에 관한 설정이다.
  • flex-start : 시작점에서부터 정렬한다. (기본값)
  • flex-end : 끝점에서부터 정렬한다.
  • center : 아이템을 부모자의 가운데에 정렬한다.
  • space-between : 처음 아이템은 시작점에 붙고 마지막 아이템은 끝점에 붙인 상태에서 나머지 아이템들을 사이 정렬한다. (좌우기준)
  • space-around : 모든 아이템들의 좌우 여백 값을 통일한다.
aline-items 아이템들의 상하 여백을 정렬한다.
(부모자의 높이가 지정되어 있고 아이템들은 부모자 높이값보다 양이 적을 때 사용함)
  • strech : 아이템들의 위아래를 맞춘다. (기본값)
  • flex-start : 아이템들을 위로 붙인다. (원래 블럭의 박스 구조임)
  • flex-end :아이템들을 아래로 붙인다.
  • center : 아이템을 부모 기준으로 가운데 정렬한다. (자손자가 하나 있을 때 사용하기 좋음)
  • space-between : 처음 아이템은 시작점에 붙고 마지막 아이템은 끝점에 붙인 상태에서 나머지 아이템들을 사이 정렬한다. (상하기준)
  • space-around : 모든 아이템들의 상하 여백 값을 통일한다.
flex-direction
  • column : 위에서 아래로 수직 정렬한다.
  • row : 왼쪽에서 오른쪽으로 수평 정렬한다.
reverse를 활용하여 역방향이 가능하다. (column-reverse와 row-reverse가 존재함)
flex-wrap 기본적으로 flex는 자손자를 한 줄에만 표시하려고 하므로 자손자의 크기는 무시하고 공간에 가둔다.
줄바꿈을 이루어 표기하려면 wrap으로 묶어준다. flex-wrap : wrap
종류 - wrap, no-wrap(기본값), wrap-reverse(역방향)이 있다.

flex - item 속성 (자손자에 표기함)
속성명 속성값과 의미
order item의 순서를 설정한다.
음수까지 허용하며 숫자가 높을수록 뒤로 밀린다. (기본값 = 0)
flex-grow 증가너비 비율을 뜻한다.
flex-shrink 감소너비 비율로 숫자가 크면 더 많은 너비가 감소한다. (반응형일때)
flex-basis item 배분 너비를 뜻하며 기본값이 auto이며 width 값은 많이 사용하지 않는다.
flex 'grow shrink basis' 순서로 기본값은 '0 0 auto' 이다.
align-self item에게 각자 지정이 가능하며 align-item의 세분화 작업을 하며 align-item의 값을 재세팅 가능하다.
