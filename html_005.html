입력양식태그

입력양식은 사용자의 정보를 입력받거나 보내는 요소이다.
예를 들어 로그인/회원가입/검색 등이 있다.

form태그 내부에 작성하며 다양한 방식의 요소를 활용한다.

form태그 내부에 전달방식을 기재하는 method요소를 활용한다.

  • get방식 - 직접전달(오픈형) / 예시 - 당근마켓
  • post방식- 비밀전달(폐쇄형) / 예시 - 본인 외 개봉금지, 아이디나 비밀번호

입력양식태그

태그명 속성명(type="속성값") 설명
form 특성 : 보이지 않음. block태그임. 입력양식의 처음과 끝
input - lnline양식 text 글자입력양식
password 비밀번호양식 - *로 표기됨
submit 제출버튼(확인)
reset 초기화버튼(다시작성하기)
file 내 컴퓨터에 있는 폴더 열기
search 검색박스로 외관으로는 text박스와 같으나 내용을 넣으면 끝에 x표시가 나타난다.
button 버튼생성 -
checkbox 다중선택박스 - 사각형 모양
radio 단일선택버튼 - 동그라미 모양_속성의 이름을 통일
email @를 넣지 않았을 경우 입력양식 오류 나타남
number 숫자입력방식 - 다양한 속성을 함께 작성
color 컬러를 선택할 수 있는 기능
range 슬라이더바 - 숫자 범위로 나타남
datetime - local, month, week, date, time
value="이름작명" 외부에 보이는 글씨 변경
name="네이밍작성" 내부에 연결해주는 속성
min="최소값" max="최대값" step="건너뛰기" 숫자가 들어가는 경우 모두 적용
textarea cols="가로공간(글씨최대값)" rows="줄바꿈갯수" 글자공간태그로 메모 등 긴 글 작성시 사용
select multipull = 'multipull' 선택양식생성
option 셀렉트태그 내부에 선택하는 요소
optgroup option태그를 묶어주는 역할은 하면서 클릭은 안됨
fieldset 그룹된 부분을 선으로 그려서 박스를 만들어 준다.
legend fieldset 내부에 선을 뚫고 써지는 제목
