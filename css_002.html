박스속성

웹 페이지 레이아웃 시 가장 중요한 속성

  • 요소 크기 설정 - width, height
  • 영역 설정 - padding
  • 외곽선 설정 - border
  • 여백 설정 - margin

작성법

  • 전체 적용 - 숫자와 단위를 함께 하나만 작성함.
  • 각자 적용 - 숫자와 단위를 4번 써서 작성하며 순서는 위, 오른쪽, 아래, 왼쪽이다. 작성 시 쉼표를 쓰지 않고 띄어쓰기를 통해 작성한다.
  • 세트 적용 - 위 아래/ 왼쪽 오른쪽 숫자와 단위를 2번 써서 작성한다. 앞에가 위 아래, 뒤에가 왼쪽 오른쪽 각자 적용과 마찬가지로 띄어쓰기를 통해 작성한다.
  • 부분 적용
    • margin-top : 위쪽 여백
    • margin-bottom : 아래쪽 여백
    • margin-left : 왼쪽 여백
    • margin-right : 오른쪽 여백
    • padding-top : 위쪽 영역
    • padding-bottom : 아래쪽 영역
    • padding-left : 왼쪽 영역
    • padding-right : 오른쪽 영역

border설정방법

박스의 테두리를 말하며 패딩 바깥쪽에 위치한다.
색상, 굵기, 모양으로 설정한다.

  • border-color : 색상기재
  • border-width : 숫자와 단위
  • border-style
    • none - 선을 없앰
    • solid - 실선
    • dotted - 동그라미 점으로 점선만들기
    • dashed - 긴 사각형 점선
    • double - 실선 두줄
    • groove - (액자형) 테두리가 파인 것처럼 그려짐
    • ridge - (액자형) 테두리가 튀어나온 것처럼 그려짐 _ groove 반대 스타일
    • inset - (액자형) 내부요소가 파인 것처럼 그려짐
    • outset - (액자형) 내부요소가 튀어나온 것처럼 그려짐 _ inset 반대 스타일

작성법 - color, width, style을 한번에 쓸 수도 있고 각자 쓸 수도 있다.(순번에 관계없음)
border : red 2px solid(2px너비의 빨간색 실선)

background(배경설정)

배경색이나 배경이미지를 삽입할 때 사용한다.
많은 속성들을 한번에 기재할 수도 있고 별도로 기재할 수도 있다.

  • background-color
    • 영문이름으로 작성 : red, black, white 등
    • hex코드로 작성 : #ffffff, #000000, #fff(ffffff를 줄인 것임) 등
      (반복으로 들어갈 경우 레드에서 하나 그린에서 하나 블루에서 하나씩 작성가능)
    • RGB수치값 작성 : RGB(00,20,50)
    • RGB와 투영도를 같이 작성 : RGBA(0,200,50,0.5) _ A는 알파값

  • background-image : url(이미지 경로 작성)

  • background-repeat : 배경이미지 또는 컬러를 반복한다.
    • repeat : 가로, 세로형으로 모두 패턴처럼 깔림(ex.바닥타일)
    • repeat-x : 가로방향으로만 깔림(ex. 벽지)
    • repeat-y : 세로방향으로만 깔림
    • no-repeat : 반복없음

  • background-size : 배경이미지의 크기를 태그사이즈대비 변경하고자 할 때 사용
    • contain : 이미지의 비율을 유지하면서 영역을 꽉 채움
    • cover : 배경이미지가 짤리더라도 화면의 영역을 꽉 채움

  • background-position : top, center, left 등 위치를 지칭하는 단어로도 사용 가능하고 숫자와 단위로도 이동가능하다.
    왼쪽 상단(모서리)에서부터 거리가 멀어지고 앞에 수가 오른쪽으로 이동하고 다음의 수가 아래쪽으로 이동한다. ex _ 20px 60px

  • background-attachment : fixed로 바닥에 붙이는 효과

그라디언트를 활용하여 배경색 또는 패턴을 제작할 수 있다.

  • linear-gradient(각도, 시작색깔, 마지막색깔)
  • radial-gradient(시작색깔, 마지막색깔)
  • patten : 직선 그라데이션의 수직 시작점을 % 값으로 잡아 옆에 여백을 만들어 반복시켜 적용한다.
  • 그라디언트헌트 바로가기
