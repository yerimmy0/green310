CSS 레이아웃

css를 활용하여 공간을 나누듯 배치를 하기 위해 여러 속성을 사용한다.


가시속성

display _ 태그가 화면에 보이는 방식을 지정한다.

  • none : 화면에서 안보이며 공간도 함께 사라진다. 또한 웹 접근성으로도 확인할 수 없다.
  • inline : 특별한 공간의 값을 가지고 있지 않다. width, height 등의 값을 지정할 수 없다.
    margin, padding의 경우 요소를 기준으로 하기 때문에 좌우는 적용이 되나, 상하는 적용이 되지 않는다.
  • inline-block : inline과 block의 중간 역할이며 두 속성의 모든 성질을 가지고 있다.
  • block : 가로 값 화면 기준 100%로 적용(자동 줄바꿈 기능이 있음)

float 속성

자동 줄바꿈의 성질이 있는 블럭을 한 줄에 2개 이상의 요소를 배치하고자 할 때 사용한다.
사전적 의미로 '물이나 공중에 뜨다' 라는 의미로 블럭 요소를 강제로 띄워서 화면에 좌우로 배치한다.

  • left : 왼쪽에 요소를 띄워서 나열한다.
  • right : 오른쪽에 요소를 띄워서 나열한다.

flex 속성

태그를 수평, 수직으로 자유롭고 유연하게 변동할 수 있는 기능이다.

  • container : 부모요소를 부르는 명칭으로 사용한다.
  • items : 자식요소를 부르는 명칭이다.

Container 속성 - display, flex-flow(묶음기능), flex-derection(수평/수직을 표현), justify-content(아이템 수평 정렬 시 사용), aline-item(아이템 수직 정렬 시 사용)

items 속성 - flex, grow(증가너비), shrink(감소너비), basis(크기배분), align-self(각자조정)

자손자에만 오롯이 적용되며 후손자에는 적용이 되지 않는다.

positon (요소를 자유롭게 배치하는 레이아웃)

여러 요소를 순서와 상관없이 화면에 가로 세로 좌표값을 기준으로 배치한다.
특정 영역 안에서 위치 조절을 자유롭게 할 수 있어 유연한 레이아웃이 가능하다.

  • relative : 적용된 요소의 현재 위치값을 기준으로 상대 위치를 지정한다. (기준값_부모요소에 주로 사용하며 또는 겹쳐져 있는 아이들을 z-index값을 적용할 때 사용)
  • absolute : 부모 요소의 특정 구간을 기준으로 절대 위치를 지정한다.(움직이는 요소)
    부모요소에 relative를 지정하지 않을 경우 window(화면)를 기준으로 움직인다.
  • fixed : 웹 브라우저를 기준으로 절대 위치를 지정한다. (화면기준으로 보임)
